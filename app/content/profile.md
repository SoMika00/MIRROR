## About this website (MIRROR)

MIRROR is Michail's portfolio and a live demonstration of his engineering
approach. It runs a fully API-first AI stack: chat completions via the xAI
Grok API, hybrid retrieval with SQLite FTS5 (BM25) and optional API
embeddings, on a small VPS with Docker and Caddy. No model weights run on the
server - a deliberate cost/latency/operability trade-off documented on the
Technical Choices page. Earlier versions ran a fully sovereign stack (Phi-4
via llama.cpp, BGE-M3 embeddings, Qdrant) on a 64 GB server; the migration
story and its trade-offs are part of the portfolio itself.
