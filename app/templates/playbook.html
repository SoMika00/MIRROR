Machine Learning

The goal is strong baselines, reproducible experiments, and metrics you can trust. I treat classical ML as a first-class tool: fast, interpretable, and often the best production ROI.

Foundations

  • Probability & stats: distributions, Bayes rule, uncertainty
  • Optimization: gradients, regularization, early stopping
  • Bias/variance: how to decide between data vs model complexity
  • Calibration: reliability curves, thresholding, decision costs

Algorithms

  • Linear/Logistic regression: robust baselines, interpretability
  • Tree models: XGBoost/LightGBM, leakage checks
  • Clustering: segment discovery, embeddings-first workflows
  • Dim reduction: PCA/UMAP for debugging and comms

Evaluation

  • Metrics: PR-AUC vs ROC-AUC, F1, cost-sensitive metrics
  • Validation: stratification, time splits, leakage prevention
  • Experimentation: A/B tests, guardrails, confidence
  • Monitoring: drift, slices, data quality checks

Production checklist

  • Reproducibility: pinned features, deterministic pipelines
  • Latency: p95 targets, batch vs online decisions
  • Observability: structured logs, traceability, dashboards
  • Failure modes: fallbacks, safe defaults, rollbacks

Deep Learning

Deep learning becomes critical when representation learning wins: language, vision, audio, multimodal. I focus on the fundamentals that transfer across domains and the system constraints that decide whether it ships.

Core building blocks

  • Backprop: stability, exploding/vanishing gradients
  • Normalization: LayerNorm, RMSNorm
  • Regularization: dropout, weight decay, data augmentation
  • Training ops: mixed precision, gradient clipping, checkpoints

NLP / LLMs

  • Tokenization: BPE/Unigram, context packing
  • Attention: causal masking, KV cache
  • Fine-tuning: LoRA/QLoRA, eval and regressions
  • RAG: retrieval, reranking, citations, prompt injection

Vision

  • Backbones: CNN vs ViT, patches, positional encodings
  • Tasks: OCR, document understanding, detection
  • Metrics: mAP/IoU, CER/WER (OCR)
  • Data: augmentation, imbalance, labeling strategy

Systems & constraints

  • Latency: batching vs streaming, TTFT, p95
  • Memory: quantization, KV cache sizing, fragmentation
  • Throughput: concurrency limits, queueing, backpressure
  • Safety: guardrails, refusal behavior, abuse prevention
Transformer overview
Shipping AI (LLMOps / MLOps)

Shipping is where most projects fail. I focus on evaluation, observability, and controlled iteration so quality improves without surprises.

Evaluation & regression

  • Golden sets: targeted examples + edge cases
  • Rubrics: correctness, groundedness, style, safety
  • RAG eval: retrieval recall, citation accuracy, reranker gains
  • Release gates: prevent silent regressions

Operations

  • Telemetry: TTFT, p95 latency, token usage
  • Rollbacks: prompt/model version pinning
  • Cost: caching, context budgets, fallbacks
  • Security: prompt injection mitigation, secrets
